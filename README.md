# Portfolio Demo Hub

Portfolio Demo Hub — это сайт-портфолио для демонстрации AI/automation проектов.

Он показывает:
- AI-консультанта для сайта;
- Smart Lead Form / Cost Calculator;
- страницы проектов;
- запуск demo;
- demo admin;
- контактную форму;
- аналитику активности пользователей.

## Что внутри

Основной проект находится в папке:

```text
portfolio-demo-hub/
```

Подробная документация:

```text
portfolio-demo-hub/README.md
```

## Технологии

* FastAPI
* PostgreSQL
* SQLAlchemy
* Docker Compose
* Nginx
* Jinja2
* HTML/CSS/JS

## Demo projects

* Universal AI Site Consultant
* Smart Lead Form / Cost Calculator

## Быстрый запуск

```bash
cd portfolio-demo-hub
cp .env.example .env
docker compose up --build
```

## Основные ссылки

```text
http://localhost/
http://localhost/projects
http://localhost/launch/ai-site-consultant
http://localhost/launch/smart-lead-form

```
